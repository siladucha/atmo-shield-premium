## 🧪 Как Проверить

1. **Запустить приложение**
   ```bash
   flutter run
   ```

2. **Сделать измерение**
   - Quick Mode (30s) для BPM
   - Accurate Mode (60s) для HRV

3. **Проверить результаты**
   - BPM должен быть 60-90 (не 125!)
   - HRV должен рассчитаться (из 15+ IBIs)

4. **Сравнить с Apple Watch** (опционально)
   - BPM: разница ±10
   - HRV: разница ±20ms
